# **App Name**: Centryx Vision

## Core Features:

- Unified Login: Single login for smart lighting and CCTV monitoring platform.
- Two-Factor Authentication: Implement 2FA using email verification code for enhanced security.
- Real-time Monitoring: Display real-time data from CCTV and smart lighting systems.
- Dark/Light Mode Toggle: Users can switch between light and dark themes for optimal viewing experience.
- Design Preview Area: Provide a section to showcase system features through visualizations.

## Style Guidelines:

- Dark Mode: Background - Dark navy blue (#1A202C) for a professional and modern feel. Light Mode: White.
- Dark Mode: Primary color - White. Light Mode: Orange (#FFB347), echoing the company logo.
- Accent color: Soft lavender (#E6E6FA), an analogous hue with brightness variation to create balance.
- Body and headline font: 'Inter' sans-serif for clean, modern readability.
- Split-screen layout with login on the right and feature display on the left for balance and informative presentation.
- Use minimalist line icons to represent smart lighting and CCTV functionalities.
- Subtle transitions and animations on login and mode switching to improve UX.
